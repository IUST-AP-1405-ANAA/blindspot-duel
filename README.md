# 🎯 BlindSpot Duel

[![Build Executables](https://github.com/IUST-AP-1405-ANAA/blindspot-duel/actions/workflows/build-release.yml/badge.svg)](https://github.com/IUST-AP-1405-ANAA/blindspot-duel/actions/workflows/build-release.yml)
[![Python Version](https://img.shields.io/badge/python-3.13-blue.svg)](https://www.python.org/downloads/)
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

بازی **BlindSpot Duel** یک بازی دو نفره رقابتی در محیط گرافیکی (توسعه‌یافته با Pygame) است که با هدف تمرین مفاهیم بنیادین برنامه‌نویسی شی‌گرا (OOP) و طراحی تجربه کاربری جذاب خلق شده است.

در این بازی هیچ‌یک از بازیکنان مکان نشانگر (Aim) خود را نمی‌دانند! نشانگرها تنها با اولین شلیک نمایان می‌شوند که چالش و هیجان بازی را دوچندان می‌کند.

---

## 🌟 ویژگی‌ها و قابلیت‌ها

- **نشانگر پنهان (Blind Crosshair):** مکانیک اصلی بازی مبتنی بر آزمون و خطا و نشان‌گیری غریزی.
- **سیستم مدیریت منابع:** مدیریت استراتژیک تیرها (Ammo) و زمان محدود.
- **سیستم امتیازدهی هوشمند:** احتساب امتیاز بر اساس فاصله از مرکز سیبل و همچنین **سیستم کومبو (Combo)** برای شلیک‌های موفق پشت‌سرهم.
- **آیتم‌های ویژه (Power-ups):** اسپاون آیتم‌هایی نظیر تیرِ اضافه، زمانِ اضافه و گلیچ (ایجاد اختلال دید برای حریف).
- **سیستم حساب کاربری و لیدربورد:** اتصال به پایگاه‌داده SQLite برای ثبت‌نام، لاگین امن و نمایش برترین بازیکنان (Leaderboard).
- **جلوه‌های بصری و صوتی (VFX & SFX):** نمایش پارتیکل‌های انفجار، متون معلق امتیازات، نمایش خون/افکت صوتی در حین برخورد و لرزش صفحه.
- **توقف بازی (Pause Menu):** قابلیت متوقف‌سازی جریان بازی بدون از دست رفتن رکوردها.

## 🎮 کنترل‌های بازی

| بازیکن / عملیات | کلیدهای حرکت | کلید شلیک |

## 🚀 نصب و اجرا (از طریق سورس‌کد)

### پیش‌نیازها
- پایتون `3.13` (یا نسخه‌های جدیدتر `3.x`)
- کتابخانه `pygame`

### مراحل اجرا
۱. مخزن پروژه را کلون کنید:
```bash
git clone https://github.com/IUST-AP-1405-ANAA/blindspot-duel.git
cd blindspot-duel
```

۲. وابستگی‌ها را نصب کنید:
```bash
pip install -r requirements.txt
```

۳. در صورت نیاز به تولید افکت‌های صوتی:
```bash
python generate_sounds.py
```

۴. بازی را اجرا کنید:
```bash
python run.py
```

## 📦 دانلود فایل‌های اجرایی (بدون نیاز به نصب پایتون)

اگر قصد دارید بازی را بدون نصب پایتون و کدها تجربه کنید، می‌توانید آخرین نسخه اجرایی ساخته شده را مستقیماً از بخش **Releases** گیت‌هاب دریافت کنید:

1. به تب **[Releases](https://github.com/IUST-AP-1405-ANAA/blindspot-duel/releases)** در مخزن گیت‌هاب مراجعه کنید.
2. جدیدترین نسخه منتشر شده (مانند `v.1.0`) را پیدا کنید.
3. در قسمت **Assets**، فایل اجرایی مربوط به سیستم‌عامل خود را دانلود کنید:
   - ویندوز: `blindspot-duel-Windows.exe`
   - لینوکس: `blindspot-duel-Linux`
   - مک: `blindspot-duel-macOS`

> **نکته:** پس از دانلود در سیستم‌عامل‌های لینوکس و مک، ممکن است نیاز باشد با دستور `chmod +x` قابلیت اجرایی را به فایل بدهید.

## 🛠 معماری و توسعه

این پروژه با رعایت کامل اصول **SOLID** و **شی‌گرایی (OOP)** توسعه داده شده است.
همچنین الگوهای طراحی نظیر **State Pattern** (جهت مدیریت صفحات و StateManager) در معماری کدهای آن پیاده‌سازی شده تا قابلیت توسعه‌پذیری پروژه حفظ شود.
